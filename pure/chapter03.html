1. 파일 내용 보기 : cat / more / tail

cat — 짧은 파일 내용 보기

  • 간단한 텍스트 파일을 한 번에 출력하는 명령.
  • -n 옵션: 행 번호 출력.
cat [옵션] 파일명
cat -n memo.txt    # 행 번호와 함께 내용 출력

more — 한 화면씩 내용 보기

  • 긴 파일을 페이지 단위로 출력.
  • 다음 페이지: Space / 이전 페이지: b / 종료: q / 문자열 찾기: /문자열
  • +행번호 옵션: 지정한 행부터 출력.
more [옵션] 파일명
more +50 longfile.txt

tail — 마지막 부분 출력

  • 파일의 끝부분을 일정 행 수만큼 보여줌.
  • +행번호: 지정 행부터 끝까지 출력.
  • -숫자: 마지막 N행 출력.
  • -f: 파일이 갱신될 때마다 실시간 출력(로그 모니터링). 
tail -10 syslog.log   # 마지막 10행 출력
tail -f access.log    # 로그 실시간 보기

[실습 3-1] 파일 내용 보기와 검색

  1. 작업 디렉토리 생성: mkdir ~/file_test
  2. 파일 복사: cp /etc/passwd .
  3. 파일 내용 보기: cat passwd
  4. 내용 검색: grep "root" passwd
  5. 다음 검색: n
  6. 종료: q

2. 파일 복사 : cp

  • 파일이나 디렉토리를 복사하는 명령.
  • 주요 옵션:
    • -i: 덮어쓰기 전에 확인.
    • -r: 디렉토리 전체 복사.
cp [옵션] 원본 대상
cp file1 file2             # 파일 복사
cp file1 file2 dir/        # 여러 파일을 디렉토리에 복사
cp -r dir1 dir2            # 디렉토리 복사

3. 파일 삭제 : rm

  • 지정한 파일 또는 디렉토리를 삭제.
  • 주의: 삭제한 파일은 복구할 수 없음!
  • 옵션:
    • -i: 삭제 전 확인.
    • -f: 강제 삭제.
    • -r: 디렉토리 전체 삭제.
rm file1
rm -i file2
rm -r old_dir

4. 파일 이동 및 이름 변경 : mv

  • 파일이나 디렉토리의 이름을 변경하거나 다른 디렉토리로 이동.
  • 옵션:
    • -i: 덮어쓰기 전에 확인.
    • -f: 강제 이동.
mv [옵션] 원본 새이름
mv file1 file2        # 이름 변경
mv file1 dir/         # 파일을 다른 디렉토리로 이동
mv dir1 dir2          # 디렉토리 이름 변경

[실습 3-2] 이동/삭제하기

  1. 작업 디렉토리 만들기: mkdir ~/move_test
  2. 파일 이동: mv test1.txt move_test/
  3. 디렉토리 이름 바꾸기: mv move_test backup_dir
  4. 디렉토리 삭제: rm -r backup_dir

5. 파일 링크 : ln

  • 한 파일에 대해 또 다른 이름(링크)을 생성.
  • 종류:
    • 하드 링크: 동일 파일의 또 다른 이름. 둘 중 하나를 수정하면 동일 내용.
    • 심볼릭 링크(-s): 경로를 가리키는 별도 파일 (윈도우의 바로가기와 유사). 
ln file1 file2        # 하드 링크
ln -s file1 link1     # 심볼릭 링크

복사(cp)와 링크(ln)의 차이

cp ln
별도의 독립된 복사본 생성 원본 파일을 참조 (공유)
둘 중 하나 수정해도 다른 파일에 영향 없음 둘 중 하나 수정하면 내용이 함께 변경됨
독립 작업에 사용 공동 관리 목적에 사용

6. 파일 생성 및 수정 시간 변경 : touch

  • 지정한 파일이 없으면 새로 생성, 있으면 접근/수정 시간을 변경.
  • 주요 옵션:
    • -a: 접근 시간만 변경.
    • -m: 수정 시간만 변경.
    • -c: 존재하지 않으면 생성하지 않음.
    • -r ref_file: 기준 파일의 시간으로 설정.
    • -t [[CC]YY]MMDDhhmm[.ss]: 시간을 직접 지정.
touch newfile.txt
touch -a existing.txt
touch -t 202510091800 sample.txt

요약

  • cat, more, tail로 파일 내용을 확인할 수 있다.
  • cp로 복사, rm으로 삭제, mv로 이동 및 이름 변경을 수행한다.
  • ln으로 하드/심볼릭 링크를 생성해 파일을 공유하거나 관리할 수 있다.
  • touch로 파일을 새로 만들거나 수정·접근 시간을 변경할 수 있다.
