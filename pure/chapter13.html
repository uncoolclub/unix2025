1. 셸 스크립트의 개요

셸 스크립트는 셸 명령들을 파일로 작성하여 반복 작업을 자동화하는 프로그램이다. 배시는 리눅스와 유닉스에서 가장 널리 사용되는 셸이다.

스크립트 기본 형태

#!/bin/bash
echo "Hello World"
  • #!/bin/bash — 스크립트를 해석할 셸 지정
  • chmod +x file.sh 로 실행 권한 부여

2. 변수와 인자

사용자 변수

name="Alice"
echo $name

환경 변수

echo $PATH
export MYVAR=123

위치 매개변수

  • $0 — 스크립트 이름
  • $1, $2 ... — 명령행 인자
  • $# — 인자 개수
  • $@ — 전체 인자
echo "첫 번째 인자: $1"
echo "전체 인자: $@"

읽기 입력

read name
echo "입력값: $name"

3. 제어문

조건문 (if)

if [ $num -gt 10 ]; then
  echo "10보다 큼"
else
  echo "10 이하"
fi

조건 검사(test 또는 [ ])

  • -f 파일 존재 여부
  • -d 디렉토리 여부
  • -e 존재 여부
  • -gt, -lt, -eq 숫자 비교
  • =, != 문자열 비교

case 문

case $1 in
  start) echo "시작";;
  stop) echo "중지";;
  *) echo "알 수 없는 명령";;
esac

반복문 (for)

for i in 1 2 3
do
  echo $i
done

반복문 (while)

count=1
while [ $count -le 5 ]
do
  echo $count
  count=$((count+1))
done

4. 연산

산술 연산

a=$((3+5))
echo $a

문자열 연산

str="Hello"
echo ${#str}      # 문자열 길이

5. 함수

함수 정의

myfunc() {
  echo "함수 실행"
}

myfunc

함수 인자

add() {
  echo $(($1 + $2))
}

add 3 5

6. I/O 및 리다이렉션

표준 입출력

  • 0 — stdin
  • 1 — stdout
  • 2 — stderr

파일로 출력

echo "hello" > out.txt      # 덮어쓰기
echo "world" >> out.txt     # 이어쓰기

입력 리다이렉션

cat < input.txt

파이프

ls -l | grep txt

7. 스크립트 작성 예시

#!/bin/bash
if [ $# -lt 1 ]; then
  echo "사용법: $0 파일명"
  exit 1
fi

if [ -f $1 ]; then
  echo "파일 존재"
else
  echo "파일 없음"
fi

요약

  • 셸 스크립트는 반복 작업 자동화에 유용하다.
  • 변수, 인자, 조건문, 반복문이 핵심 구성 요소다.
  • 입출력 리다이렉션과 파이프를 이용해 명령을 조합할 수 있다.
  • 함수를 사용하면 재사용성과 구조화가 향상된다.
