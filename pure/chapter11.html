1. 파일 정보 수집 — wc

wc 명령어는 파일의 라인 수, 단어 수, 문자 수를 확인할 때 사용한다.

옵션

  • -l — 라인 수
  • -w — 단어 수
  • -c — 바이트 수
  • -m — 문자 수

예시

wc test.txt
wc -l test.txt
wc -w test.txt

2. 파일 정렬 — sort

sort 명령어는 텍스트 파일 내용을 정렬한다.

주요 옵션

  • -r — 역순 정렬
  • -u — 중복 제거
  • -n — 숫자 정렬
  • -d — 사전순 정렬
  • -t 문자 — 필드 구분자 지정
  • -k N — N번째 필드 기준 정렬
  • -o 파일명 — 결과를 파일로 저장

예시

sort scores.txt
sort -r scores.txt
sort -t " " -k 2 students.txt
sort -o result.txt scores.txt

3. 파일 분할 — split

split 명령어는 큰 파일을 여러 조각으로 분할한다.

옵션

  • -l N — N줄씩 분할
  • -b 크기 — 지정한 바이트 단위로 분할

예시

split -l 1000 data.txt
split -b 1m logfile.txt

4. 중복 삭제 — uniq

uniq 명령어는 연속된 중복 라인을 제거한다.

옵션

  • -c — 중복 횟수 표시
  • -d — 중복된 행만 출력
  • -u — 중복되지 않은 행만 출력

예시

uniq names.txt
sort names.txt | uniq -c
uniq -d names.txt

5. 필드 추출 — cut

cut 명령어는 파일에서 특정 필드 또는 문자를 잘라낸다.

옵션

  • -c 위치 — 특정 문자 위치
  • -f 필드 — 특정 필드
  • -d 문자 — 필드 구분자 지정

예시

cut -c 1-5 text.txt
cut -d " " -f 1,3 data.txt

6. 파일 연결 — paste

paste 명령어는 여러 파일을 줄 단위로 붙인다.

옵션

  • -d 문자 — 필드 구분자 지정
  • -s — 수평으로 붙이기

예시

paste file1 file2
paste -d ":" file1 file2
paste -s file1

7. 파일 덤프 — dd

dd 명령어는 데이터를 변환하거나 특정 블록 단위로 복사할 때 사용한다.

주요 옵션

  • if=파일 — 입력 파일
  • of=파일 — 출력 파일
  • bs=N — 블록 크기
  • conv=lcase — 소문자로 변환
  • conv=ucase — 대문자로 변환

예시

dd if=input.txt of=output.txt bs=1m
dd if=source.txt of=lower.txt conv=lcase

요약

  • wc — 파일 라인·단어·문자 정보 확인
  • sort — 파일 정렬 및 중복 제거
  • split — 파일 분할
  • uniq — 연속 중복 라인 제거
  • cut — 특정 필드 또는 문자 추출
  • paste — 여러 파일을 줄 단위로 결합
  • dd — 파일 변환 및 블록 단위 복사
