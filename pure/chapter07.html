1. 파일 내용 검색: grep / egrep / fgrep

grep 명령어는 파일 내용에서 특정 패턴이나 문자열을 검색하는 도구이다. 정규 표현식을 지원하여 복잡한 패턴 검색이 가능하다.

grep 명령어

Global Regular Expression Print의 줄임말로, 파일에서 특정 패턴과 일치하는 행을 찾아 출력한다.

기본 문법

grep [옵션] 패턴 [파일...]

주요 옵션

  • -i — 대소문자 구분하지 않음
  • -v — 패턴과 일치하지 않는 행 출력
  • -n — 행 번호와 함께 출력
  • -c — 일치하는 행의 개수만 출력
  • -l — 패턴이 포함된 파일명만 출력
  • -w — 단어 단위로 정확히 일치하는 경우만 검색
  • -x — 행 전체가 패턴과 일치하는 경우만 검색

사용 예시

grep "root" /etc/passwd        # passwd 파일에서 "root" 검색
grep -i "error" log.txt        # 대소문자 무시하고 "error" 검색
grep -n "function" script.sh   # 행 번호와 함께 "function" 검색
grep -c "warning" *.log        # 모든 .log 파일에서 "warning" 개수 출력

egrep (확장 grep)

확장 정규 표현식을 사용할 수 있는 grep이다. grep -E와 동일하다.

  • + — 앞 문자의 1회 이상 반복
  • ? — 앞 문자의 0회 또는 1회
  • | — OR 연산자
  • () — 그룹화
egrep "colou?r" file.txt       # "color" 또는 "colour" 검색
egrep "(cat|dog)" animals.txt  # "cat" 또는 "dog" 검색

fgrep (고정 문자열 grep)

정규 표현식을 사용하지 않고 고정 문자열만 검색한다. grep -F와 동일하다. 특수 문자를 문자 그대로 검색할 때 유용하다. 

fgrep "*.txt" file.txt         # "*.txt" 문자열 그대로 검색
fgrep -f patterns.txt data.txt # patterns.txt의 각 행을 패턴으로 검색

2. 파일 검색: find

find 명령어는 파일 시스템에서 파일과 디렉토리를 검색하는 강력한 도구이다. 파일명, 크기, 수정 시간, 권한 등 다양한 조건으로 검색할 수 있다.

기본 문법

find [검색경로] [검색조건] [동작]

주요 검색 조건

  • -name 패턴 — 파일명으로 검색 (대소문자 구분)
  • -iname 패턴 — 파일명으로 검색 (대소문자 무시)
  • -type 타입 — 파일 타입으로 검색
  • -size 크기 — 파일 크기로 검색
  • -mtime 일수 — 수정 시간으로 검색
  • -perm 권한 — 파일 권한으로 검색
  • -user 사용자 — 소유자로 검색
  • -group 그룹 — 그룹으로 검색

파일 타입

  • f — 일반 파일
  • d — 디렉토리
  • l — 심볼릭 링크
  • b — 블록 장치
  • c — 문자 장치

크기 지정

  • +크기 — 지정 크기보다 큰 파일
  • -크기 — 지정 크기보다 작은 파일
  • 크기 — 정확히 지정 크기인 파일
  • 단위: c(바이트), k(킬로바이트), M(메가바이트)

기본 사용 예시

find . -name "*.txt"           # 현재 디렉토리에서 .txt 파일 검색
find /home -type d -name "test"  # /home에서 test 디렉토리 검색
find . -size +1M               # 1MB보다 큰 파일 검색
find . -mtime -7               # 7일 이내 수정된 파일 검색
find . -perm 755               # 권한이 755인 파일 검색

동작 옵션

  • -print — 검색 결과 출력 (기본값)
  • -exec 명령 {} \; — 검색된 파일에 명령 실행
  • -ok 명령 {} \; — 실행 전 확인 후 명령 실행
  • -delete — 검색된 파일 삭제

논리 연산자

  • -and 또는 -a — AND 연산 (기본값)
  • -or 또는 -o — OR 연산
  • -not 또는 ! — NOT 연산

3. 명령어 검색: which

which 명령어는 PATH 환경변수에 설정된 경로에서 실행 파일의 위치를 찾는다. 셸에서 명령어를 실행할 때 실제로 사용되는 파일의 경로를 보여준다.

기본 문법

which [옵션] 명령어

주요 옵션

  • -a — PATH에서 찾은 모든 실행 파일 경로 출력

사용 예시

which ls                       # ls 명령어의 위치 검색
which python                   # python 실행 파일 위치 검색
which -a gcc                   # 모든 gcc 실행 파일 위치 검색

type 명령어

명령어의 타입을 확인한다 (내장 명령어, 별칭, 함수, 실행 파일).

type ls                        # ls 명령어 타입 확인
type cd                        # cd는 셸 내장 명령어
type -a echo                   # echo의 모든 정의 확인

요약

  • grep, egrep, fgrep은 파일 내용에서 패턴을 검색하는 강력한 도구이다.
  • find는 파일 시스템에서 다양한 조건으로 파일과 디렉토리를 검색할 수 있다.
  • which는 PATH에서 명령어의 실행 파일 위치를 찾는다.
  • 이러한 검색 도구들을 조합하면 효율적인 파일 관리와 시스템 관리가 가능하다.
