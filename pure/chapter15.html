1. DOS의 개요

DOS(Disk Operating System)는 명령어 기반 운영체제로, 파일 관리, 프로그램 실행, 하드웨어 초기화를 담당한다.

  • 명령 프롬프트에서 문자 기반으로 명령을 입력하여 사용
  • 파일 확장자 중심의 파일 관리(.exe, .com, .bat 등)

2. 기본 명령어

1) 화면 및 디렉토리 관련

cls               # 화면 지우기
dir               # 디렉토리 목록
dir /p            # 페이지 단위 출력
dir /w            # 넓게 출력

2) 디렉토리 이동

cd 폴더명
cd ..             # 상위 디렉토리
cd \              # 루트 디렉토리

3) 디렉토리 생성/삭제

mkdir data
rmdir data

3. 파일 관리 명령

파일 복사

copy a.txt b.txt
copy *.txt backup\

파일 삭제

del  a.txt
del *.tmp

파일 이름 변경

ren old.txt new.txt

파일 내용 보기

type memo.txt
more memo.txt

4. 실행 파일과 배치 파일

실행 파일 종류

  • .exe — 실행 파일
  • .com — 간단한 실행 파일
  • .bat — 배치(일괄 처리) 파일

배치 파일 예시

@echo off
echo 백업 시작
copy *.txt backup\
echo 완료

5. 디스크 관리 명령

디스크 상태

chkdsk

디스크 포맷

format A:

부팅 디스크 생성

sys A:

6. 환경 설정

환경 변수

set PATH=C:\bin;%PATH%
set TEMP=C:\TEMP

AUTOEXEC.BAT / CONFIG.SYS

DOS 부팅 시 자동 실행되는 환경 설정 파일

7. 검색 및 필터링

파일 검색

dir /s keyword*

파이프 사용

dir | more

8. 도움말

help
help command

요약

  • DOS는 텍스트 기반 운영체제로 단순하지만 빠르고 가벼운 파일 관리가 가능하다.
  • dir, cd, copy, del 등은 기본적인 파일·폴더 관리 명령이다.
  • 배치 파일(.bat)을 이용하면 반복 작업을 자동화할 수 있다.
  • chkdsk, format 등으로 디스크 관리가 가능하다.
